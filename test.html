Instructions

  • Soduku is a classic puzzle game that takes place on a 9x9 grid which is sub-divided into nine 3x3 grids. The objective is to fill each row, coumn, and 3x3 grid exactly with numbers 1-9 (not necessarily in numerical order). Consequently, no row, column may have any repeated digits. The grid will have some of the values filled in for you, but more of the grid will be blank if you pick a higher difficulty.
